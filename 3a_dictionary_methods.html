As we can see, there are differences within documents in terms of negative and positive sentiment matches. Importantly, it seems that document length also varies significantly. Lets normalize the sentiment scores by total words and then calculate the difference between positive and negative proportion of words.

docvars(kcna_dfm, "prop_negative") <- as.numeric(kcna_sent_dfm[,1] / ntoken(kcna_dfm)) # add proportion of negative words to original dfm
docvars(kcna_dfm, "prop_positive") <- as.numeric(kcna_sent_dfm[,2] / ntoken(kcna_dfm))
docvars(kcna_dfm, "net_sentiment") <- docvars(kcna_dfm, "prop_positive") - docvars(kcna_dfm, "prop_negative") # calculate net sentiment

Targeted dictionary analysis

How about a more targeted analysis, say the sentiment of words around the term nuclear? We can use the tokens_select and window argument to perform this targeted dictionary analysis. First, we will construct a tokens object from the original corpus object. Then the documents will become a window of terms +/- 15 tokens.

toks <- tokens(corpus_kcna, 
               remove_numbers = TRUE,
               remove_punct = TRUE,
               remove_symbols = TRUE,
               remove_separators = TRUE,
               remove_url = TRUE,
               include_docvars = TRUE)

toks_key <- tokens_keep(toks, 
                         pattern = 'nuclear',
                         valuetype = "glob", 
                         window = 15) # get window of words around keyword

toks_key <- dfm(toks_key) # create dfm of tokens object

toks_key_lsd <- toks_key %>% 
  dfm_lookup(dictionary = data_dictionary_LSD2015[1:2]) %>%
  dfm_group(group = week, fill = TRUE)

# plot net sentiment
n_toks <- ntoken(dfm_group(toks_key,
                     group = docvars(toks_key, 'week'))) # get number of weekly tokens

plot((toks_key_lsd[,2] - toks_key_lsd[,1]) / n_toks,
      type = 'l', ylab = "Net Sentiment", xlab = '', main = 'Net sentiment around "nuclear"', xaxt = "n") # plot difference in share of positive and negative terms
axis(1, seq_len(ndoc(toks_key_lsd)), ymd("2015-01-01") + weeks(seq_len(ndoc(toks_key_lsd)) - 1))
grid()
abline(h = 0, lty = 2)

How about another term like Japan? We can use the same code as above but just swap nuclear for japan in the pattern argument for tokens_keep.

We see a consistent negative sentiment in windows that include the term japan over the sample time period, although there are weeks with net positive sentiment.
